Slugfeast is a HTML5 game from the Arduino class during my Interaction Design studies.

After picking up the basics of Arduino coding, we decided to use it as a game controller. We knew it would be a huge challenge since Arduino doesn't serially communicate to the browser. This is when I learned the basics of node.js and socket.io.

To allow Arduino to communicate with the browser, socket.io was one of the very few options that were available.

A non-Arduino version of this game can be found here. Use space to shoot, and up/down arrows to control the angle.
