Bagasell

Design & projektledning

One2tel i Malmö behövde ett lättanvänt men kraftfullt system för orderhantering.

Ihop med en före detta Microsoft-anställd, en utvecklare vid namnet Doug Gale, skapade jag ett kraftfullt orderhanteringssystem för företag i telekombranschen.

Jag var ansvarig för hela projektet, samt UI designen.

Vi stötte på flertalet problem på vägen under detta projekt, och jag lärde mig verkligen vikten av att göra ett ordentligt förarbete innan man påbörjar ett projekt av denna omfattningen.

Att alltid planera för potentiella problem i förväg, men ändå bibehålla ett öppet sinne och vara flexibel för förbättringar under projektets gång.
Det var receptet som gjorde att vi kunde realisera detta projektet, trots att jag arbetade ihop med en utvecklare som satt på andra sidan av Atlanten.

Vi anpassade Bagasell exklusivt för telemarketingföretag. Det vill säga företag vars anställda har i uppgift att ringa kalla samtal till sina kunder, för att förlänga eller nyteckna abonnemang.

Eftersom det är en marknad där säljarna oftast gör 1-20 säljavslut dagligen, var det viktigt att vi anpassade systemet så att de snabbt och enkelt kunde placera ordrar, utan att behöva tappa fokus från det som är viktigast för dom.

Vi skapade ett system där de ansvariga enkelt kunde förinställa allt i förväg, för att underlätta så mycket som möjligt för säljarna.

Ett sätt att placera ordrar för ett framtida datum var ett måste.

Detta är ett av de många problem vi fick lösa på vägen. Vissa kunder hade abonnemang som löpte ut en månad efter att försäljningen skedde, så det var viktigt att säljarna kunde registrera dessa ordrar trots detta. Denna lösning löste också problemet att säljarna inte kunde "stjäla" varandras ordrar av misstag.

Vi byggde också ett enkelt system för kommentarer, där säljarna och kundtjänsten kan kommunicera på ett smidigt sätt.

Orderhistoriken tillåter säljarna och kundtjänsten att snabbt få en överblick över vad som är nytt, vad som väntar, vad som är registrerat etc.

Denna sektion tillåter dessutom användarna att snabbredigera sina ordrar, lägga till fler kommentarer, få en enkel inblick av vilken kund de har sålt till, och mycket mer.

Vi skapade också AJAX-drivna datumintervall, söksystem, filter m.m.

Bagasell tillhandahåller ett fullmatat notifieringssystem där användarna på ett smidigt sätt kan hålla koll på alla viktiga händelser.
